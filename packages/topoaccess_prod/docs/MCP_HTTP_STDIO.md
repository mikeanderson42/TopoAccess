# MCP, HTTP, And Stdio

TopoAccess publishes compatible local tool manifests for HTTP, MCP-like tool registration, and JSON-line stdio.

Exact lookup forbids model fallback. Post-edit validation is read-only. Provenance fields are required for audited outputs.

Generate:

```bash
python packages/topoaccess_prod/scripts/topoaccess_generate_manifests.py --profile default --out release/topoaccess_prod_v37
```
