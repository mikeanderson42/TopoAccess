# Install

Wrapper path:

```bash
python packages/topoaccess_prod/scripts/topoaccessctl.py install --cache cache/topoaccess_v21
python packages/topoaccess_prod/scripts/topoaccessctl.py start --cache cache/topoaccess_v21
python packages/topoaccess_prod/scripts/topoaccessctl.py status --cache cache/topoaccess_v21
```

Native Windows Task Scheduler or WSL systemd application remains manual/operator controlled. Use the V28/V29 generated manual instructions and keep rollback available.
