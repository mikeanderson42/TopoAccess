# TopoAccess v1.0.0-rc1

TopoAccess is a local repo-intelligence sidecar for coding agents.

This release candidate includes:

- Public package layout under `packages/topoaccess_prod`.
- Model-free public CI.
- Codex, generic CLI, HTTP, stdio, and documentation-first harness integration assets.
- Exact lookup tool-only policy.
- Category-gated model fallback for planning/synthesis/troubleshooting categories.
- Apache-2.0 license and Michael A. Anderson creator/project owner metadata.

V39.1 fixes the public CI cache-package export issue and adds release asset plus branch-protection guidance.
