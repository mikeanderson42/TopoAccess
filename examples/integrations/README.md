# Integration Examples

These files are small, source-controlled examples for harness setup. They replace generated release-folder artifacts in the public repository.

- `AGENTS.md`: agent guidance for Codex-style tools.
- `claude_hooks/settings.example.json`: safe Claude Code hook example.
- `cursor_rules/topoaccess.mdc`: Cursor rule example.
- `schemas/openapi.json`: OpenAPI-style schema.
- `schemas/mcp_like_manifest.json`: MCP-like manifest.
- `schemas/stdio_schema.json`: stdio tool schema.
- `schemas/tool_schema.json`: generic TopoAccess tool schema.
- `repomap/`: small token-budgeted repo-map examples.

Generated release archives and checksums are attached to GitHub Releases rather than committed to the source tree.
